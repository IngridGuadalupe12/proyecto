por ejemplo unos de los lenguajes de JavaScript

JavaScript es uno de los lenguajes de programacion mas usados para crear paginas web dinamicas.Permite a los programadores implementar funciones complejas en las paginas web para añadirles mas interactividad. Una pagina dinamica es aquella que incopora efectos,como texto que aparece y desaparece,animaciones,acciones que se activan al pulsar botones ventanas con mensajes de aviso al usario.
